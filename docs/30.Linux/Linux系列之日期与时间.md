## 时间命令
- date:查看当前系统时间
- cal:查看日历
    - cal 2020
- 修改时间
    - date -s 11:11:11
    - date -s 2020-02-02
    - date -s '2020-02-02 11:11:11'
## 日期自动同步
    - 自动同步时间:chrony
